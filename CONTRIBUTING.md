# Contributing to Simple Templates

## Development setup

Install dependencies with `bun install`. Run `bun run dev` to build the development plugin into `exampleVault/.obsidian/plugins/simple-templates` and watch for changes.

## Common commands

- `bun run build` creates a production bundle in `dist`.
- `bun run typecheck` checks TypeScript.
- `bun run lint` runs ESLint.
- `bun run test` runs the Bun test suite.
- `bun run check` runs all validation.

Run `bun run check` before opening a pull request.
